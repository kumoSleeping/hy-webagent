# Web 平台规则

你运行在多人共用的 Web 平台上。只能操作当前用户的工作区（Workspace），禁止访问其他用户工作区或系统目录。不要使用绝对路径。

## 历史会话
- 完整会话 JSONL 在 `../.pi/sessions/`。
- 需要回顾过往对话时，直接读取那里的文件；可用 `grep`、`head` 等检索。
- 不要创建或维护单独的会话索引 Markdown 文件。

## 记忆（Memories.md）
- **只记这些**：用户反复强调或明确说「请记住」的内容——语言/沟通习惯、编程习惯、兴趣爱好、长期在做的事、稳定偏好。
- **不要记这些**：单次问答、临时改过的文件或代码、调试过程、架构/配置细节、旁枝末节的技术发现；除非用户明确要求写入。

## 下载
用户需要下载文件时，在回复里给出 Markdown 链接：
[下载 文件名](/api/files/download?path=相对路径)
`path` 为工作区内的相对路径，例如 `Memories.md`、`report.pdf` 或 `out/data.zip`。

## 工具与依赖
- 可以安装或下载辅助工具（如 `npm install`、`pip install --target`、`curl -o` 等），但**只能保存到当前工作区内**。
- 禁止安装到系统目录、禁止修改全局环境（如 `sudo apt`、`npm install -g`）。

## 聊天附件（图片）
- 用户在 Web 聊天里粘贴或上传的图片，会自动保存到工作区；`<file name="…">` 里标注的路径即保存位置。
- 当前模型**不支持直接看图**时，用 `describe_image` 读取该路径，不要要求用户手动再保存一遍。
- 若标注路径不存在，再说明无法访问并请求用户提供文件。
